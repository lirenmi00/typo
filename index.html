typo.css

typo.css is a typography stylesheet to make your content readable. Current page is using typo.css.

typo.css is not a reset stylesheet.

It is designed for readable article content. If you are looking for a reset css, you are watching the wrong repository.

Installation

Install with bower:

$ bower install typo

If you don't use bower, you can just grab the css file from GitHub. There is no dependency of this project.

API

For readable content, wrapper them under the .typo class, and everything would be OK now:

<div class="typo">
            <h1>Heading</h1>
            <p>Paragraph of contents.</p>
            ...
        </div>

Tags

typo.css supports these commonly used tags.

Tag Name Description
h1 - h6 headings for title
p paragraph
a anchor links
strong b emphasis in bold style
em i emphasis in italic style
img image
figure figure wrapper for images
figcaption figcaption in figure
hr separator
blockquote style for quotes
ul ol li ordered and unordered list
pre code block
code tt inline code
table tables
iframe embed iframe style
mark mark some text

Headings

Headings are tags <h1>, <h2>, <h3>, <h4>, <h5> and <h6>. But the most commonly used tags are <h1>, <h2>, and <h3>.

Emphasis

Emphasis works on something important or valuable. <b> and <i> are not designed for this purpose, you should always use <strong> and <em>.

typo.css

typo.css 是一个优化阅读体验的网页排版样式表。当前页面使用了typo.css，还有热前端。

typo.css 不是复位样式表。

它是为优化文章阅读体验而设计的。如果你正在寻找一个复位样式表，你看错代码库啦。

安装

用 bower 安装:

$ bower install typo

如果你不使用bower，你可以从GitHub获取CSS文件。这个项目没有其他依赖。

API

对于可读的内容，使用.typo类包裹它们，然后一切都OK了：

<div class="typo">
            <h1>标题</h1>
            <p>段落内容。</p>
            ...
        </div>

标题

标题包括 <h1>, <h2>, <h3>, <h4>, <h5> 和 <h6>标签。 但最常用的标签是 <h1>, <h2>, 和 <h3>。

强调

强调用在 重要的 或者 有价值 的内容上。 <b> and <i> 标签不是为此而设计的，你应该总是使用 <strong> 和 <em>标签。
